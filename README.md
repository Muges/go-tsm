# TSM

Package tsm implements several real-time time-scale modification methods, i.e. algorithms that change the playback speed of an audio signal without changing its pitch.

Read the article [A Review of Time-Scale Modification of Music Signals](http://www.mdpi.com/2076-3417/6/2/57) by Jonathan Driedger and Meinard MÃ¼ller for more details about each methods.
